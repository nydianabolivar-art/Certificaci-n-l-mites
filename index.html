1 ¿Qué es un Límite?

El concepto fundamental del cálculo explicado de manera intuitiva

📌 Definición Intuitiva

Un límite describe el valor al que se acerca una función cuando su variable independiente se aproxima a un punto específico. No importa si la función llega o no a ese valor, lo que importa es hacia dónde tiende.

limx→a f(x) = L

Se lee: "El límite de f(x) cuando x tiende a a es igual a L".

Esto significa que conforme x se acerca cada vez más a a (sin necesariamente llegar), los valores de f(x) se acercan cada vez más a L.

🔑 Analogía para entenderlo

Imagina que caminas hacia una puerta. El límite es la puerta misma: describes hacia dónde te diriges, no si cruzas la puerta o no. Puedes acercarte infinitamente a la puerta por la izquierda o por la derecha. Si ambos caminos te llevan al mismo lugar, el límite existe.

📐 Definición Formal (Epsilon-Delta)

Para todo ε > 0 (épsilon, qué tan cerca quieres estar del límite), existe un δ > 0 (delta, qué tan cerca debe estar x de a) tal que:

Si 0 < |x − a| < δ, entonces |f(x) − L| < ε

En palabras simples: puedo hacer f(x) tan cercano a L como quiera, siempre que x esté suficientemente cerca de a.

💡 Tip Especial
El límite NO depende del valor de f(a). La función puede no estar definida en x = a, o tener un valor diferente a L en ese punto, y el límite aún puede existir. Lo que importa es el comportamiento alrededor de ese punto.

↔️ Límites Laterales

Se puede estudiar qué pasa cuando x se acerca desde un solo lado:

Límite por la izquierda: limx→a⁻ f(x) = L₁
Límite por la derecha: limx→a⁺ f(x) = L₂

Regla fundamental: El límite bilateral existe si y solo si ambos límites laterales existen y son iguales:

limx→a f(x) = L ⟺ L₁ = L₂ = L
⚠️ Error Común
Muchos estudiantes confunden f(a) con limx→a f(x). Recuerda: f(a) es el valor de la función en a; el límite es hacia dónde se acerca la función cerca de a. Pueden ser iguales (función continua) o diferentes.

🧪 Experimenta: Observa el límite en acción

Función: f(x) = (x² − 4) / (x − 2). Observa qué pasa cuando x se acerca a 2. ¡La función no está definida en x = 2, pero el límite sí existe!

1.00

2 Propiedades de los Límites

Las reglas algebraicas que simplifican tu trabajo

📋 Propiedades Fundamentales

Si limx→a f(x) = L y limx→a g(x) = M, entonces:

1. limx→a [f(x) ± g(x)] = L ± M  (Suma/Resta)
2. limx→a [f(x) · g(x)] = L · M  (Producto)
3. limx→a [f(x) / g(x)] = L / M, si M ≠ 0  (Cociente)
4. limx→a [c · f(x)] = c · L  (Constante)
5. limx→a [f(x)]ⁿ = Lⁿ  (Potencia)
6. limx→a ⁿ√f(x) = ⁿ√L, si L ≥ 0  (Raíz)

📌 Límites Especiales que Debes Memorizar

limx→0 sin(x)/x = 1

limx→∞ (1 + 1/x)x = e ≈ 2.71828

limx→0 (eˣ − 1)/x = 1

limx→0 ln(1+x)/x = 1
💡 Tip Especial — Sustitución Directa Primero
Siempre intenta sustituir directamente el valor de x primero. Si obtienes un número definido, ¡ese es el límite! Solo cuando obtienes una forma indeterminada (como 0/0, ∞/∞, etc.) necesitas aplicar técnicas adicionales.

🚫 Formas Indeterminadas

Cuando la sustitución directa produce uno de estos resultados, necesitas otra estrategia:

0/0  ·  ∞/∞  ·  0·∞  ·  ∞−∞  ·  0⁰  ·  1∞  ·  ∞⁰

Estas formas no son respuestas. Son señales de que necesitas manipular algebraicamente la expresión o usar técnicas como factorización, racionalización, o la Regla de L'Hôpital.

3 Procedimiento Paso a Paso

Algoritmo sistemático para resolver cualquier límite

🗺️ Mapa de Decisión para Resolver Límites

1
Sustituir directamente el valor de x en la función. Si obtienes un número definido → ¡Ese es tu límite! Terminaste.
2
¿Obtuviste 0/0? → Intenta: factorizar, racionalizar, simplificar fracciones complejas, o identidades trigonométricas.
3
¿Obtuviste ∞/∞? → Divide numerador y denominador por la mayor potencia de x. O usa L'Hôpital.
4
¿Obtuviste k/0 (k≠0)? → El límite es ±∞ o no existe. Analiza los signos con límites laterales.
5
Regla de L'Hôpital: Si tienes 0/0 o ∞/∞, puedes derivar numerador y denominador por separado: lim f(x)/g(x) = lim f'(x)/g'(x)

🔧 Técnica 1: Factorización

Ejemplo: limx→3 (x² − 9) / (x − 3)

1
Sustituir x=3: (9−9)/(3−3) = 0/0 → Indeterminado
2
Factorizar el numerador: x²−9 = (x+3)(x−3)
3
Simplificar: (x+3)(x−3)/(x−3) = x+3
4
Sustituir x=3: 3+3 = 6 ✓

🔧 Técnica 2: Racionalización

Ejemplo: limx→0 (√(x+4) − 2) / x

1
Sustituir x=0: (√4−2)/0 = 0/0 → Indeterminado
2
Multiplicar por el conjugado: numerador y denominador por (√(x+4) + 2)
3
Numerador se convierte en: (x+4) − 4 = x
4
Simplificar: x / [x(√(x+4)+2)] = 1/(√(x+4)+2)
5
Sustituir x=0: 1/(√4+2) = 1/4 = 0.25 ✓

🔧 Técnica 3: Regla de L'Hôpital

Si limx→a f(x)/g(x) produce 0/0 o ∞/∞, entonces:

limx→a f(x)/g(x) = limx→a f'(x)/g'(x)

Ejemplo: limx→0 sin(x)/x

1
Sustituir x=0: sin(0)/0 = 0/0 → Aplica L'Hôpital
2
Derivar: f'(x) = cos(x), g'(x) = 1
3
limx→0 cos(x)/1 = cos(0) = 1 ✓
⚠️ Cuidado
L'Hôpital SOLO se aplica cuando hay 0/0 o ∞/∞. Verificar siempre que la condición se cumpla antes de derivar. ¡Aplicarla sin verificar es el error más frecuente!

🔧 Técnica 4: Límites al Infinito (polinomios racionales)

Para limx→∞ P(x)/Q(x) donde P y Q son polinomios:

• grado P < grado Q → Límite = 0
• grado P = grado Q → Límite = coeficiente líder P / coeficiente líder Q
• grado P > grado Q → Límite = ±∞

Truco: divide todo entre la mayor potencia de x del denominador.

Ejemplo: limx→∞ (3x² + 5x) / (2x² − 1)

1
Grados iguales (ambos grado 2)
2
Dividir todo entre x²: (3 + 5/x) / (2 − 1/x²)
3
Cuando x→∞: 5/x→0 y 1/x²→0
4
Resultado: 3/2 = 1.5 ✓
💡 Tip Especial — Resumen Rápido
Cuando no sepas por dónde empezar: (1) Sustituye. (2) Si da 0/0, busca factorizar o racionalizar. (3) Si nada funciona, L'Hôpital. (4) Si es ∞/∞ con polinomios, divide por la mayor potencia.

4 Continuidad y Límites

La conexión entre límites y funciones continuas

📌 ¿Qué es la continuidad?

Una función f(x) es continua en x = a si se cumplen TRES condiciones simultáneamente:

1. f(a) está definida
2. limx→a f(x) existe
3. limx→a f(x) = f(a)

Si cualquiera de estas condiciones falla, la función es discontinua en ese punto.

🔍 Tipos de Discontinuidad

Evitable (removible): el límite existe, pero f(a) no está definida o f(a) ≠ L. Se puede "reparar" definiendo f(a) = L.

De salto: los límites laterales existen pero son diferentes (L₁ ≠ L₂).

Esencial (infinita): al menos un límite lateral es ±∞.

💡 Tip Especial
Para ingeniería, la continuidad importa porque los modelos físicos suelen ser continuos. Una discontinuidad en tu modelo puede indicar un error, un punto de transición (cambio de fase, umbral, etc.) o una aproximación que necesita refinarse.

5 Aplicaciones por Carrera

Cómo los límites se usan en tu campo profesional

🎮 Renderizado y Gráficos 3D

En computación gráfica, los shaders usan límites para calcular iluminación suave. Cuando la distancia entre la fuente de luz y la superficie tiende a cero, la intensidad tiende a infinito; los límites permiten manejar este comportamiento y crear transiciones suaves.

limd→0 I/d² → Se acota para evitar explosión de brillo
🎮 Animación y Curvas de Bézier

Las curvas Bézier, fundamentales en diseño y animación, usan límites para garantizar suavidad. La continuidad de la curva (C0, C1, C2) depende de que los límites laterales coincidan en los puntos de control.

limt→t₀⁻ B'(t) = limt→t₀⁺ B'(t) → Continuidad C1 (tangente suave)
🎮 Compresión de Imágenes y Video

Algoritmos como JPEG usan la Transformada Discreta del Coseno (DCT), que se basa en límites de sumas infinitas. Entender convergencia de series (un tipo de límite) es esencial para entender calidad vs. tamaño de archivo.

🏭 Control de Calidad

En manufactura, los límites definen las tolerancias. Cuando un proceso de producción se refina, el error tiende a un valor mínimo alcanzable. Los límites permiten calcular ese valor teórico óptimo.

limn→∞ Error(n) = Tolerancia mínima del proceso
🏭 Optimización de Producción

El costo marginal (costo de producir una unidad adicional) es un límite: limΔq→0 ΔC/Δq = C'(q). Es la base para decidir cuántas unidades producir para maximizar ganancias.

Costo Marginal = limΔq→0 [C(q+Δq) − C(q)] / Δq
🏭 Fiabilidad de Sistemas

La probabilidad de falla de un componente a lo largo del tiempo se modela con funciones exponenciales. El límite cuando t→∞ da la probabilidad acumulada de falla, esencial para planificar mantenimiento.

💰 Interés Compuesto Continuo

El límite más famoso en finanzas: cuando la frecuencia de capitalización tiende a infinito, el monto tiende a una fórmula elegante con el número e.

limn→∞ P(1 + r/n)nt = P·ert

Esto es la base de la capitalización continua, usada en valoración de derivados financieros y modelos de crecimiento.

💰 Tasa Marginal y Derivadas

El ingreso marginal, el costo marginal, y la utilidad marginal son todos límites. La decisión financiera óptima ocurre cuando el ingreso marginal iguala al costo marginal.

Ingreso Marginal = limΔq→0 [R(q+Δq) − R(q)] / Δq = R'(q)
💰 Valor Presente de Perpetuidades

Una perpetuidad es un flujo de pagos que continúa para siempre. Su valor presente es un límite de una serie geométrica infinita:

VP = limn→∞ Σ C/(1+r)k = C/r
🌍 Elasticidad de la Demanda

La elasticidad precio de la demanda se define como un límite: mide cuánto cambia la cantidad demandada ante un cambio infinitesimal en el precio.

E = limΔP→0 [(ΔQ/Q) / (ΔP/P)] = (dQ/dP) · (P/Q)
🌍 Modelos de Saturación de Mercado

Cuando un producto entra a un mercado nuevo, la adopción sigue una curva logística. El límite cuando t→∞ da el tamaño máximo del mercado (saturación).

limt→∞ N(t) = K (capacidad del mercado)
🌍 Tipo de Cambio y Paridad

La paridad del poder adquisitivo (PPA) se analiza como un límite a largo plazo: el tipo de cambio tiende a igualarse con la relación de precios entre países cuando las barreras comerciales tienden a cero.
